Política de Privacidade

Última atualização: Janeiro de 2025

1. Quem somos

A Aura Dev Apps desenvolve aplicativos para lojistas. Contato: auradev.app@gmail.com

2. Dados coletados

  • Token OAuth da plataforma (Nuvemshop / Loja Integrada)
  • ID da loja do usuário
  • Chave de API de IA fornecida pelo próprio usuário

Nenhum dado pessoal como CPF ou senha é coletado ou armazenado.

3. Uso dos dados

  • Autenticar requisições às APIs das plataformas
  • Publicar produtos conforme solicitado
  • Gerar descrições via inteligência artificial

4. Compartilhamento

Não compartilhamos dados com terceiros. Comunicações apenas com a API da plataforma escolhida e serviços de IA.

5. Seus direitos (LGPD)

Você pode revogar o acesso pelo painel da plataforma a qualquer momento. Dados locais são removidos ao desinstalar o app.

6. Contato

auradev.app@gmail.com
