Configuração

Como conectar
sua loja em minutos

Um processo simples, feito uma única vez. Depois é só usar.

1

Baixe o app na Play Store

Faça o download gratuito do Nuvemshop Estoque App na Google Play Store no seu celular Android.

2

Acesse a tela de Configurações

Dentro do app, toque no ícone de engrenagem no menu inferior para acessar as configurações de credenciais.

3

Autorize via Nuvemshop

Toque em "Conectar com Nuvemshop". Você será redirecionado para o painel da Nuvemshop para fazer login com seu email e senha normais.

4

Autorize as permissões

Na tela da Nuvemshop, clique em "Aceitar e começar a usar". O app precisa de permissão para criar e editar produtos na sua loja.

5

Pronto! Sua loja está conectada

Você será redirecionado de volta ao app automaticamente. A conexão é permanente — só precisará repetir se trocar de conta.

Política de Privacidade

Última atualização: Janeiro de 2025

1. Quem somos

A Aura Dev Apps é responsável pelo desenvolvimento e manutenção dos aplicativos disponibilizados neste site. Nosso contato para assuntos de privacidade é: mdutra75@gmail.com.

2. Quais dados coletamos

Nossos aplicativos coletam e armazenam exclusivamente no dispositivo do usuário:

  • Token de acesso OAuth gerado pela Nuvemshop (para autenticação com a API)
  • ID da loja Nuvemshop do usuário
  • Chave de API do serviço de inteligência artificial (OpenRouter), fornecida pelo próprio usuário

Nenhum dado pessoal como nome, email, CPF ou informações financeiras é coletado ou armazenado por nossos aplicativos.

3. Como usamos os dados

Os dados coletados são utilizados exclusivamente para:

  • Autenticar requisições à API da Nuvemshop em nome do lojista
  • Publicar produtos na loja do usuário conforme solicitado pelo próprio usuário
  • Gerar descrições de produtos usando inteligência artificial

4. Compartilhamento de dados

Não compartilhamos, vendemos ou transferimos nenhum dado do usuário a terceiros. As únicas comunicações externas realizadas pelo app são com:

  • API oficial da Nuvemshop (api.nuvemshop.com.br) — para publicação de produtos
  • OpenRouter (openrouter.ai) — para análise de imagens via IA

5. Segurança

Os tokens de acesso são armazenados localmente no dispositivo do usuário usando SharedPreferences, com acesso restrito ao aplicativo. Nunca trafegamos sua senha da Nuvemshop.

6. Seus direitos (LGPD)

Você pode a qualquer momento revogar o acesso do app à sua loja através do painel da Nuvemshop (Meus Aplicativos → Desinstalar). Os dados locais são removidos ao desinstalar o aplicativo do dispositivo.

7. Contato

Para dúvidas sobre privacidade: mdutra75@gmail.com
